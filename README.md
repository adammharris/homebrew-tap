Hi! Welcome to my tap! I made this so it is easy to install my software on computers running macOS (and Linux).

## Install

```sh
brew tap adammharris/tap
brew install adammharris/tap/<formula>
```

## Formulae

- [`fig`](https://github.com/adammharris/fig) — parse, edit, and convert config files (JSON, YAML, TOML, …) while preserving comments.
- [`twig`](https://github.com/adammharris/twig) — parse, query, edit, and losslessly round-trip Djot, Markdown, HTML, and XML documents.
- [`colophon`](https://github.com/adammharris/colophon) — command-line companion for the colophon self-describing workspace library.
- [`moid`](https://github.com/adammharris/moid) — command-line companion for the moid opaque-ID minter.
